\subsection{Definition of Quantale-Enriched Category}

A \emph{quantale-enriched category} (also known as a $\mathcal{V}$-category, where $\mathcal{V}$ is a quantale) generalizes the notion of a category by enriching the hom-sets with values in a quantale. 

Let $\mathcal{V} = (V, \leq, \otimes, k)$ be a \emph{quantale}; that is, $V$ is a complete lattice with monoidal structure $(\otimes, k)$ such that for all $a \in V$, the maps $a \otimes -$ and $- \otimes a$ preserve all joins.

A \textbf{quantale-enriched category} $\mathcal{C}$ consists of:
\begin{itemize}
    \item A class of objects, $\mathrm{Ob}(\mathcal{C})$.
    \item For each pair of objects $x, y \in \mathrm{Ob}(\mathcal{C})$, a ``hom-object'' $\mathcal{C}(x, y) \in V$.
    \item For each $x \in \mathrm{Ob}(\mathcal{C})$, an element $k \leq \mathcal{C}(x, x)$ (identity axiom).
    \item For each $x, y, z \in \mathrm{Ob}(\mathcal{C})$, 
    \[\mathcal{C}(x, y) \otimes \mathcal{C}(y, z) \leq \mathcal{C}(x, z)\]
    (composition axiom).
\end{itemize}

If $\mathcal{V}$ is the Boolean quantale $\{\bot, \top\}$ with $\otimes = \wedge$ (meet), we recover the notion of a preordered set. When $\mathcal{V}$ is $([0, \infty], \geq, +, 0)$ (the quantale of extended nonnegative reals with addition and reverse order), we get generalized metric spaces.
